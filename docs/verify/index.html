What this checks
Independent verification for exported evidence.

The verifier recomputes the canonical packet hash, verifies the RS256 signature against JWKS, and checks the append-only event chain. If any one of those breaks, the page tells you exactly why.

1
Hash

Detects mutations to the canonical export artifact.

2
JWKS

Picks the correct public key from signature_kid.

3
Chain

Confirms every event points back through the audit log.
