Authorize every handoff. Prove every tool call.

This demo shows the product moment Attest is built for: a human authorizes a task, an orchestrator delegates narrower authority to a sub-agent, the tool call is enforced, and the full chain can still be verified later.

No mock dashboard theatre. Just the chain of authority, visible end to end.

What this proves

Attest is not just an audit log. It is evidence that a specific tool action came from a bounded chain of delegated authority rooted in a human instruction.

1 root credential
1 delegated child

Run the authority chain

Start with a human instruction and choose the scopes the orchestrator is allowed to hold. The child agent will receive only the minimum it needs.

The child agent in this demo receives only email:send. If the root lacks that scope, delegation fails.
